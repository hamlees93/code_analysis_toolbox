#Code Style

##Installation

```
$ gem install rubocop rubocop-rails flay
```

Copy and paste the .rubocop.yml file into the root of the project.

##Running

Run the below command from the root of the project.

```
$ ruby {absolute_path_to_caconventions.rb}
```
